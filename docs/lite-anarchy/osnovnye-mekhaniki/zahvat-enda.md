# Захват Энда

  Эндер мир по умолчанию закрыт. Открывается он каждое воскресенье в 16:00 по МСК всего на 90 минут. Можно залутать **яйцо дракона**, осколочные яйца из энда и кастомные и ванильные данжи. Чтобы попасть в Эндер мир нужно **телепортироваться через портал** на спавне `/warp end`. 

  
 **Игрок** нанёсший больше всего урона по дракону Энда получит **яйцо дракона**, которое имеет свою цену. Чтобы повысит его цену храните его в сундуке. Яйцо в последствии можно обменять на сапфиры.Яйца дракона, хранящиеся где-то в мире можно увидеть на `/warp top`.
